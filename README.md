# 🧮 Scientific Calculator

> A modern desktop-based Scientific Calculator developed using **Java Swing**, featuring an intuitive graphical user interface and comprehensive mathematical functionalities.

## 📖 Overview

This project is a desktop-based Scientific Calculator built with **Java Swing**. It provides a clean graphical interface for performing both basic and advanced mathematical operations. The application emphasizes accuracy, usability, and robust exception handling, making it suitable for educational purposes and everyday scientific calculations.

---

## ✨ Features

### 🔢 Basic Operations
- Addition
- Subtraction
- Multiplication
- Division
- Percentage

### 📐 Scientific Functions
- Sin
- Cos
- Tan
- Cot
- Sec
- Cosec

### 📊 Logarithmic & Exponential
- Log (Base 10)
- Natural Log (ln)
- Euler's Number (e)

### ⚡ Advanced Operations
- Square (x²)
- Cube (x³)
- Power (xⁿ)
- Square Root (√)
- Cube Root (∛)
- Reciprocal (1/x)
- Factorial
- Absolute Value

### 🛡 Error Handling
- Division by Zero
- Invalid Expressions
- Mathematical Exceptions
- Input Validation
